Index of modules

  • lib/EvmUtil

    The EVM Utility Canister interface.

    Used by the library for performing various evm-related computations.

  • lib/IcManagement

    The IC Management Canister interface used by the library.

    Used by the library for creating public keys and signing messages.

  • lib/NoKeyWalletLib

    Stable class adding wallet support for EVM based chains.

    Supports creating t-ecdsa key and using it to sign transactions for EVM chains. Also maintains and exposes the record of keys, nonce and the transactions by the principal.

  • lib/State

    State stored by the library.

    As a library user, you should rarely have to interact with this module directly.

  • lib/Types

    Type declarations for the library.
